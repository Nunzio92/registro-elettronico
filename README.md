# 🎓 Registro Elettronico - Corso React per Studenti
## Costruisci la Tua Prima App React da Zero!

---

## 🚀 **Benvenuto nel Corso React!**

In questo corso di **2 giorni** costruirai da zero un'applicazione web completa: un **Registro Elettronico** per studenti.

### 🎯 **Cosa Imparerai**
- ⚛️ **React Fundamentals** - Componenti, Props, State
- 🎣 **React Hooks** - useState, useEffect, useContext
- 🌐 **Context API** - Gestione stato globale
- 🛣️ **React Router** - Navigazione tra pagine
- 🎨 **Material-UI** - Componenti UI professionali
- 📡 **API Integration** - Chiamate HTTP con fetch
- 🎭 **CSS Variables** - Theming dinamico

### 🏆 **Cosa Costruirai**
Al termine avrai un'app completa con:
- ✅ Sistema di login/logout
- ✅ Dashboard con statistiche
- ✅ Navigazione multi-pagina
- ✅ Tema scuro/chiaro
- ✅ Design responsive
- ✅ Integrazione con backend

---

## 🛠️ **Setup Rapido**

### 1. **Installazione**
```bash
# Installa le dipendenze
npm install
```

### 2. **Avvio**
```bash
# Avvia frontend + backend insieme
npm run dev
```

### 3. **Accesso**
- **App**: http://localhost:3000
- **API**: http://localhost:3001
- **Login**: `studente` / `password`

## 🆘 **Hai Bisogno di Aiuto?**

### 🐛 **Errori Comuni**
1. **Errore di sintassi**: Controlla parentesi e virgole
2. **Import sbagliati**: Verifica percorsi e nomi file
3. **State non aggiornato**: Usa spread operator `{...state}`
4. **useEffect infinito**: Aggiungi dipendenze corrette

### 🔍 **Come Debuggare**
1. **Console Browser** (F12) - Vedi errori JavaScript
2. **React DevTools** - Ispeziona componenti e state
3. **Network Tab** - Controlla chiamate API
4. **Console.log** - Il tuo migliore amico!

### 💡 **Suggerimenti**
- 📖 Leggi sempre i messaggi di errore completi
- 🔄 Salva spesso e testa ad ogni modifica
- 🤝 Chiedi aiuto ai compagni o all'insegnante
- 💪 Non aver paura di sperimentare!
